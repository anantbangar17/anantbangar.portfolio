SecureVault — Zero-Knowledge Password Manager

A production-grade, zero-knowledge password manager built during my internship at Sangam (India) Limited. Sensitive data — usernames, passwords and notes — is AES-encrypted client-side using a PBKDF2-derived key before ever leaving the browser. The server stores only cipher text and never sees plaintext credentials.

  • Implemented zero-knowledge encryption architecture — AES encryption via crypto-js with PBKDF2 key derivation; backend handles only cipher text, ensuring server-side breaches expose no user data
  • Built a complete JWT-based auth system — stateless session management with bcryptjs password hashing, token signing and protected API routes in Express.js
  • Designed a fully typed React + TypeScript frontend with Vite and Tailwind CSS, integrating Axios for HTTP and Web Crypto API for secure client-side password generation
  • Architected a RESTful Node.js + Express backend with MySQL (mysql2), uuid for record IDs, dotenv for config management and cors for cross-origin handling
